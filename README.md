# 🛒 FreshMart – Desktop Management Application

> **BCA 5th Semester Project**  
> A desktop-based management application developed using **C#**, **.NET Windows Forms**, and **SQL Server** to manage products, customers, inventory, and daily business operations.

---

## 📖 Overview

FreshMart is a desktop application designed to simplify retail store management by providing an easy-to-use interface for managing products, customers, inventory, and sales records.

This project was developed as part of my **Bachelor of Computer Applications (BCA) – 5th Semester** coursework and helped me strengthen my understanding of desktop application development using C# and .NET.

---

## 🚀 Features

### 🛒 Product Management

- Add new products
- Update product information
- Delete products
- Search products

### 👥 Customer Management

## ⚙️ Getting Started

### Prerequisites

- Visual Studio
- .NET Framework
- SQL Server

### Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/freshmart-desktop-application.git
```

2. Open the solution in Visual Studio.

3. Restore the SQL Server database.

4. Update the connection string.

5. Build and run the project.
